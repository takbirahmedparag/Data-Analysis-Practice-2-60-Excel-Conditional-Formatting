# Data-Analysis-Practice-2-60-Excel-Conditional-Formatting
This repository contains my Excel practice on Conditional Formatting using Percentage and Grade columns. The goal is to highlight data using colors based on conditions. This is part of my Data Analysis Practice series (2/60).
## Project Objective
The main objective of this project is to understand how to use Conditional Formatting in Microsoft Excel to highlight important data automatically.
This practice helps to:
•	Understand Conditional Formatting rules
•	Highlight important values using colors
•	Improve data visualization in Excel
•	Build basic data analysis skills

## Questions Covered (Dataset 1)
1.	Highlight the Percentage column in Green if Percentage ≥ 80
2.	Highlight the Percentage column in Yellow if Percentage is between 50 and 79
3.	Highlight the Percentage column in Red if Percentage < 50
4.	Highlight the Grade column in Green if the student is Pass
5.	Highlight the Grade column in Red if the student is Fail

## KPI (Key Performance Indicators Dataset 1)
•	Correct use of Conditional Formatting
•	Correct rule for Percentage ≥ 80
•	Correct rule for Percentage between 50–79
•	Correct rule for Percentage < 50
•	Correct formatting for Pass and Fail
•	Proper color visualization
